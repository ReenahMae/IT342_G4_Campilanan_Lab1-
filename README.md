Project Description

This project is a Mini Authentication System that allows users to:

Register an account

Log in using valid credentials

Access a protected dashboard

Update profile settings

Log out securely

Protected pages such as Dashboard and Profile cannot be accessed without authentication.

The system follows a 3-layer architecture:

Frontend: React

Backend: Spring Boot

Database: MySQL

How to Run Backend

Start XAMPP

Start Apache

Start MySQL

Create database in phpMyAdmin

Open: http://localhost/phpmyadmin

Create database named: miniauth_db

Configure application.properties

File location:
backend/miniauth/src/main/resources/application.properties

spring.datasource.url=jdbc:mysql://localhost:3306/miniauth_db?useSSL=false&serverTimezone=UTC
spring.datasource.username=root
spring.datasource.password=

spring.jpa.hibernate.ddl-auto=update
spring.jpa.show-sql=true

server.port=8080


Run the backend

From IntelliJ:
Run MiniauthApplication.java

Or from terminal inside backend/miniauth:

mvn clean install
mvn spring-boot:run


Backend runs at:
http://localhost:8080

How to Run Web

Open terminal inside the web folder:

cd web


Install dependencies:

npm install


Start the React app:

npm start


Web runs at:
http://localhost:3000

Make sure the backend is running before logging in.

Database Setup Instructions

Database name: miniauth_db

Tables are automatically generated by Hibernate

The users table includes:

id

first_name

middle_name

last_name

email

password_hash

role

created_at

updated_at

No manual SQL import is required.
